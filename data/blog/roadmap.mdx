I took some time to plan out a potential development roadmap, and here is the outline I came up with:

> [!IMPORTANT]
> This roadmap is a work in progress and may change as development evolves. None of the upcoming releases have set deadlines. As a solo developer working on this project in my free time, I wish I could provide more concrete dates, but my availability depends on my work and personal schedule.

---

## Version 0.2a 

- Introduce Gym Leaders 3–6 (4 new Gym Leaders)
- Add more Pokémon (amount TBD)
- Add more Fakémon (amount TBD)
- Potential implementation of custom scripts
- Create and verify test events for gameplay flow and debugging
- Begin documenting development workflow for contributors

---

## Version 0.3a

- All 8 Gym Leaders now available
- Add more Pokémon (amount TBD)
- Add more Fakémon (amount TBD)
- Continue creating and verifying test events
- Continue documentation for development workflow

---

## Version 0.4a

- Add Elite Four
- Add more Pokémon (amount TBD)
- Add more Fakémon (amount TBD)
- Ongoing test event verification and debugging
- Continue contributor documentation

---

## Version 0.5a

- Add Champion
- Implement Hall of Fame animation
- Introduce endgame content
- Add more Pokémon (amount TBD)
- Add more Fakémon (amount TBD)
- Continue test event development
- Update documentation as needed

---

## Version 0.6a and beyond

- Expand endgame content
- Add more Pokémon (amount TBD)
- Add more Fakémon (amount TBD)
- Finalize test events and polish gameplay
- Refine contributor documentation
- Focus on feature polish, balance adjustments, and bug fixing

---

## Version 1.0b – Beta Build

- First full Beta release
- Feature-complete with all core content implemented

---

## Version 1.0 – Stable Release

- Final release version
- Fully polished and balanced
- Bug-free (as close as possible) and ready for wide play
